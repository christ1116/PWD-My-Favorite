Animasi dan Series

Media yang saya tonton ada banyak, tetapi saya hanya akan pilih 3, yaitu:

1. The Walking Dead

Saya suka bagaimana series ini mencerminkan dunia apabila dunia dipenuhi dengan zombie atau bisa dibilang drama tetapi dengan zombie

  • Karakter yang memorable seperti Rick, Daryl dan Glenn
  • Latar yang bagus
  • Saya suka ceritanya

2. Girls Last Tour

Anime yang menceritakan dua perempuan yang menjelajah kota terbengkalai di apocalypse

  • Saya suka Karakter nya
  • Animasi dan latar yang bagus
  • Mempunyai makna yang dalam yaitu tetap bahagia walaupun dunia tidak baik-baik saya
  • Banyak salju

3. Regular Show

Kartun komedi dengan cerita absurb dan hilarious

  • Saya suka alur cerita nya yang sangat absurb
  • Animasi nya unik
  • Karakter-karakter yang sangat memorable seperti Mordecai, Rigby dan Benson

Game

Saya juga banyak main game tapi saya hanya akan sebutkan 3 saja, yakni:

1. Semua Game Retro

Saya sangat suka game retro karena game nya sangat simple, walaupun hampir semua game retro itu susah dan saya sangat menghargai orang-orang yang membuat game retro yang secara tidak langsung membuat game menjadi populer bahkan di masa sekarang

  • Game nya simple
  • Game nya masih seru walaupun sudah lama

2. Doom

Termasuk game jadul tetapi saya masukkan ke kategori sendiri dikarenakan game ini adalah game favoritku sepanjang masa

  • Game FPS
  • Bisa pasang mod atau map pack yang dibuat komunitas yang masih kuat sampai sekarang sejak game ini rilis di tahun 1993
